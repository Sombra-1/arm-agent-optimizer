# Final report

The final HTML report is generated exclusively from `report-data.json` and the verified Passport; it never recomputes selection. It embeds CSS and accessible SVG with no remote dependencies or network requests.

Sections cover the outcome, baseline comparison, quality gate, fastest rejected candidate, funnel, service rate, P95, RSS, quality table, Pareto frontier, drift sentinel, hardware, runtime/KleidiAI, model/workload provenance, settings, reproduction, methodology, limitations, and hashes. Unavailable metrics are omitted or labelled unavailable, never plotted as zero.

Synthetic reports carry an unavoidable top banner.
