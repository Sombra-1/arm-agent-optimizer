# Judging evidence map

**Challenge track: Cloud AI**

AArchTune targets Arm64 cloud/server CPU inference and improves both the optimization workflow and the reliability of the selected deployment configuration.

| Criterion | Evidence |
|---|---|
| Technical implementation | Version-aware flag inspection, bounded planning/screening, isolated server lifecycle, real validators, process metrics, drift sentinel, Pareto analysis, canonical Passport hashes |
| Developer experience | One-command workflow, native stages, safe resume, concise CLI summaries, self-contained report, selected profile and scripts |
| Potential impact | Reduces manual Arm inference tuning while preventing speed-only correctness regressions |
| Wow factor | Fastest-rejected quality card, explainable funnel/Pareto visualization, verifiable Optimization Passport |

Real quantitative claims must link to a validated Arm64 Passport. Synthetic output demonstrates behavior only.
